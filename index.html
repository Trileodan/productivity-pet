What's buildable

✓ Live lock-screen timer + pet — a Live Activity (iOS) / ongoing notification (Android) shows remaining time & pet state on the lock screen during a session, updating frequently — unlike normal widgets, which are refresh-limited. ✓ Depletion notification — local notifications fire even when the app is backgrounded; no server needed. ✓ Phone budget + 10s drain — pure timer logic. ✓ All RPG + campaign systems — levels, evolutions, classes, maze, tokens: ordinary app logic, authored server-side. ✗ Sensing phone while screen off — still impossible; budget model sidesteps it.
